DocCharConvert

Ekaya keyboard

Ekaya is a smart input method for Windows based on KMFL and compatible with Keyman. It includes a Myanmar Unicode 5.1 keyboard.

Graphite OpenOffice Extension

The Graphite OpenOffice.org Extension enables the Graphite Font Rendering functionality in OpenOffice 3.2 and above to be enabled and disabled and makes it easy to use Graphite Font Features.
