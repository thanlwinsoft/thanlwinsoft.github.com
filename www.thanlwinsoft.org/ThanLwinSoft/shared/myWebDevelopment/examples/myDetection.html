Myanmar Unicode Test

You are free to use this on your own website without charge or waranty provided you add a link of acknowledgement to www.thanlwinsoft.org.

To use this script on your site you need to:

  • include a few scripts e.g. unzip myWebDevelopment.zip in the /styles directory
  • add an onload function

မြန်မာစကား

...
<meta content="text/html; charset=UTF-8" http-equiv="content-type" />
<script type="text/javascript" src="/styles/myDetection.js"></script>
<script type="text/javascript" src="/styles/xmlRequest.js"></script>
</head>
<body  onload="myUnicode.initParse('/styles/myWebDevelopment/');" >

...

Change the styles directory as appropriate for your website. The Text2Image conversion relies on the images' files being in a subdirectory of /styles/ or whatever you specify as the argument to initParse. If you are using Firefox or Opera, then it will switch to embedded SVG rather than images, since these are more versatile. (The SVG only example demonstrates this explicitly). For more details View the Source HTML, CSS and Javascript for instructions on how to use in your own pages.

If the conversion fails, Myanmar [မြန်မာ မြန်မာ] text should automatically be converted to images. If your browser supports Myanmar [မြန်မာ] Unicode you can still force the conversion.

If you want support for Myanmar Unicode in text input fields you can use the Myanmar Keyboard.
