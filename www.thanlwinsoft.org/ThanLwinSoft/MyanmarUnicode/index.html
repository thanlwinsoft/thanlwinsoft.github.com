Myanmar Unicode

A Guide

This guide explains how you can install the existing Myanmar Unicode implementations for the Myanmar (Burmese) language.

This guide discusses Myanmar Implementations on both Linux and Windows. A brief mention is also made about the Mac, but the author has no experience of Myanmar Unicode on that platform.

A Myanmar Unicode Quick Start Guide is now available for users. However, if you have problems, you may still want to read the more detailed pages. This guide uses Myanmar Unicode 5.1 to represent Myanmar characters, so you may wish to follow the quick start guide to install a Myanmar Unicode font before reading further.

Please read on for the background technical information.
